À propos de ce site

ruby-lang.org est maintenu par des membres de la communauté Ruby. Le code source est disponible sur github. Quelques uns des outils utilisés :

  • Jekyll pour la génération statique
  • Octopress pour ses plugins et son thème Classic
  • Thor pour l’outil shell de maintenance
  • Compass pour le développement Sass
  • Pygments et pygments.rb pour la coloration syntaxique
  • et quelques autres encore !
